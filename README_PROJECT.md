# MegaHashAr - Децентрализованная платформа взаимопомощи

Современный 3D веб-сайт для проекта MegaHashAr, построенный с использованием React, Three.js и TypeScript.

## 🚀 Особенности

- ✨ Интерактивная 3D графика с Three.js
- 🎨 Современный дизайн в стиле неоморфизм
- 📱 Полностью адаптивный дизайн
- ⚡ Быстрая загрузка благодаря Vite
- 🎭 Плавные анимации с CSS и Framer Motion
- 🌐 Многоязычная поддержка (планируется)

## 📋 Требования

- Node.js 18+
- npm или yarn

## 🛠️ Установка

```bash
# Клонировать репозиторий
git clone https://github.com/malishomen/megahashar.git

# Перейти в директорию проекта
cd megahashar

# Установить зависимости
npm install
```

## 🎯 Запуск проекта

### Режим разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для production

```bash
npm run build
```

### Предварительный просмотр production сборки

```bash
npm run preview
```

## 🎨 Технологический стек

- **Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **3D Graphics**: Three.js, React Three Fiber
- **Animation**: Framer Motion
- **Styling**: CSS3, CSS Modules

## 📱 Разделы сайта

1. **Главная** - Hero секция с анимированными счетчиками
2. **Миссия** - Цели и философия проекта
3. **Whitepaper** - Документация и технические детали
4. **Дорожная карта** - План развития проекта
5. **Аналитика** - SWOT анализ и метрики
6. **FAQ** - Часто задаваемые вопросы

## 🌟 Ключевые функции

- Интерактивная 3D модель логотипа "Нексус Верификации"
- Анимированные счетчики статистики
- Интерактивная временная шкала дорожной карты
- SWOT-анализ с динамическими элементами
- Аккордеон FAQ с плавными анимациями

## 🔧 Конфигурация

Основные настройки находятся в:
- `vite.config.ts` - конфигурация Vite
- `tsconfig.json` - настройки TypeScript
- `src/styles/globals.css` - глобальные стили и CSS переменные
